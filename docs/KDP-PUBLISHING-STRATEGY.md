# KDP Publishing Strategy: Dr. Lesa Mulligan's Cookbook

## Overview

This document outlines the strategy for publishing "Dr. Lesa Mulligan's Cookbook: A Collection of Family Recipes" as a hardcover book through Amazon Kindle Direct Publishing (KDP).

---

## Book Specifications

### Recommended Format
- **Trim Size:** 8" x 10" (ideal for cookbooks with recipes and photos)
- **Interior:** Premium color (70# paper) for recipe photos
- **Binding:** Hardcover (case laminate - cover art printed directly on cover)
- **Page Count:** ~180-200 pages (min 75, max 550 for KDP hardcover)

### Alternative Options
- **Budget Option:** 7" x 10" trim, standard color
- **Economy Option:** Paperback 8.5" x 11" (lower production cost)

## Pricing Strategy

### Production Costs (Estimated)
- Hardcover 8" x 10", 200 pages, premium color: ~$13-15 per copy
- Minimum list price to earn royalty: ~$25-30

### Recommended Pricing
- **Hardcover:** $34.99 - $39.99
- **Paperback:** $24.99 - $29.99 (if offered)
- **Royalty Rate:** 60% minus printing costs (expanded distribution) or higher for Amazon-only

## Step-by-Step Publishing Process

### 1. Prepare Manuscript
- [x] Compile all 150 recipes from JSON files
- [ ] Format in Word/InDesign to 8" x 10" trim
- [ ] Add chapter dividers with category headers
- [ ] Include recipe card photos (optional - increases page count and cost)
- [ ] Export as PDF (300 DPI minimum for images)

### 2. Create Cover
- Design front cover, spine, and back cover as single PDF
- Minimum spine width: 0.5" for 200 pages (requires 79+ pages for spine text)
- Cover template available from KDP after setting trim size and page count

### 3. KDP Account Setup
1. Go to kdp.amazon.com
2. Create account or sign in
3. Click "Create" > "Hardcover"
4. Enter book details:
   - Title: "Dr. Lesa Mulligan's Cookbook"
   - Subtitle: "A Collection of 150 Family Recipes"
   - Author: Dr. Lesa Mulligan
   - Categories: Cookbooks > Regional & International > American
   - Keywords: family recipes, comfort food, home cooking, traditional recipes, cookbook

### 4. Upload Files
- Upload interior PDF (manuscript)
- Upload cover PDF
- Preview with KDP Previewer tool
- Fix any formatting issues

### 5. Set Pricing & Distribution
- Choose expanded distribution for wider availability
- Set price (hardcover typically $30-40 for color cookbooks)
- Select territories (worldwide recommended)

### 6. Publish
- Submit for review (24-72 hours typically)
- Book goes live on Amazon

## ISBN Options

### Option A: Free KDP ISBN
- Free, assigned by Amazon
- Can only be used for Amazon distribution
- Shows "Independently published" as publisher

### Option B: Your Own ISBN
- Purchase from Bowker ($125 for 1, $295 for 10)
- Use your own imprint name
- Required for expanded bookstore distribution

**Recommendation:** Start with free KDP ISBN, purchase your own later if successful.
