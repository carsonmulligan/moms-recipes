# Mom's Recipes

A collection of family recipes digitized from handwritten recipe cards.

## View Online

Visit the GitHub Pages site to browse all 70+ recipes with a dropdown selector.

## Structure

```
├── index.html              # Recipe viewer (dropdown selector)
├── thanksgiving/
│   ├── desserts/           # Thanksgiving dessert recipes
│   └── savory-tings/       # Thanksgiving side dishes
└── misc/                   # All other recipes
```

Each recipe has:
- Original recipe card photo (JPG)
- JSON file with structured data (title, ingredients, instructions)
