# Mom's Thanksgiving Recipes

A collection of family recipes digitized from handwritten recipe cards.

## View Online

Visit the GitHub Pages site to browse all recipes with a dropdown selector.

## Recipes Included

### Desserts
- Buttermilk Chess Pie
- No Bake Cookies
- Peanut Butter Ball (Dip)
- Peanut Butter Balls (Rice Krispies)
- Pecan Pie
- Pumpkin Pie

### Sides & Salads
- Corn Casserole
- Cranberry Salad
- Fruit Salad with Cannoli Cream
- Garlic Mashed Potatoes
- Green Beans Amandine
- Marinated Vegetable Salad
- Sweet Potato Casserole
- Taffy Apple Salad
- Thanksgiving Turkey Dressing

### Sweets & Treats
- Caramels
- Caramel Apple Crumble Pie

## Structure

```
thanksgiving/
├── desserts/
│   ├── [recipe-name].jpg    # Photo of recipe card
│   └── [recipe-name].json   # Transcribed recipe data
└── savory-tings/
    ├── [recipe-name].jpg
    └── [recipe-name].json
```

Each recipe has:
- Original recipe card photo (JPG)
- JSON file with structured data (title, ingredients, instructions)
